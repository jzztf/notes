# 将linux安装到USB移动设备

推荐工具：[Universal USB Installer](https://www.pendrivelinux.com/universal-usb-installer-easy-as-1-2-3/)

```bash
# usb内目录
.
├── boot
│   └── grub
│       └── menu.lst
├── initrd.lz
├── LOST.DIR
├── lubuntu-18.04-desktop-amd64.iso
├── System Volume Information
│   └── WPSettings.dat
├── uui
│   ├── grldr
│   ├── ldlinux.sys
│   └── syslinux.cfg
└── vmlinuz
```
